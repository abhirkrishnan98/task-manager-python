# 📝 Task Manager with User Authentication (Python CLI)

A simple command-line based Task Manager built using Python.  
This application allows users to register, login, and manage their personal tasks with persistent file storage.

---

## 🔧 Features

- 🔐 User Registration & Login (with password hashing)
- ➕ Add Tasks
- 📋 View Tasks
- ✅ Mark Tasks as Completed
- 🗑️ Delete Tasks
- 📁 File-based storage (no database required)
- 👤 User-specific task management

---

## 🛠️ Tech Stack

- Python
- File Handling
- Hashlib (for password security)
